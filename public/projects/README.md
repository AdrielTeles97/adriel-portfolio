# Project Images

Adicione as imagens dos projetos nesta pasta:

## Arquivos necessários:

- project-1.jpg
- project-2.jpg
- project-3.jpg
- project-4.jpg

## Especificações recomendadas:

- **Resolução**: 800x600px ou maior
- **Aspect ratio**: 4:3
- **Formato**: JPG ou WebP
- **Tamanho máximo**: 200KB por imagem

## Dicas:

- Use screenshots dos seus projetos reais
- Mockups de dispositivos funcionam bem
- Mantenha consistência visual entre as imagens

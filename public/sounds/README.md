# Sound Files

Para o sistema de sons funcionar, adicione os seguintes arquivos nesta pasta:

## Arquivos necessários:

1. **ambient.mp3** - Som ambiente de fundo (looping)
    - Recomendação: música ambient suave, lo-fi, ou som de natureza
    - Volume base: 15%
    - Duração: 1-3 minutos (vai loopar)

2. **hover.mp3** - Som ao passar o mouse sobre elementos
    - Recomendação: click suave, "whoosh" curto
    - Duração: 0.1-0.3 segundos

3. **click.mp3** - Som ao clicar em botões
    - Recomendação: click mecânico, "pop"
    - Duração: 0.1-0.2 segundos

4. **transition.mp3** - Som durante transições de página/seção
    - Recomendação: "swoosh", transição suave
    - Duração: 0.3-0.5 segundos

## Onde encontrar sons gratuitos:

- [Freesound.org](https://freesound.org)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Pixabay](https://pixabay.com/sound-effects/)
- [Zapsplat](https://www.zapsplat.com)

## Dicas:

- Mantenha os arquivos pequenos (< 500KB cada, exceto ambient)
- Use formato MP3 para melhor compatibilidade
- Normalize o volume dos sons para consistência
